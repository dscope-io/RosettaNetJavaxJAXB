Guideline Information

2: standardName.GlobalAdministeringAuthorityCode

User Notes: In the case of a RosettaNet-compliant message, the only allowed value is "RosettaNet".

Entity Instances
RosettaNet: RosettaNet Consortium

 

3: standardVersion.VersionIdentifier

User Notes: When standardName.GlobalAdministeringAuthorityCode is equal to "RosettaNet" then standardVersion.VersionIdentifier MUST carry the version number of the RNIF specification. For a message compliant with RNIF 02.00, this value MUST be "V02.00".
